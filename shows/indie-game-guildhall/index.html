Indie Game Guildhall 访谈 Future Suit Games：在开发日常中寻找快乐游戏

Indie Game Guildhall2026/06/08 12:47:05

本期 Indie Game Guildhall 邀请到了 Future Suit Games 的联合创始人 Abram 和 Christie，探讨独立游戏开发者在日常磨砺中寻找快乐的话题。Abram 讲述了自己从美工设计师到桌游设计、从失败 Kickstarter 到重拾创作热情、最终通过 Indie Game Academy 找到联合创始人的历程。Christie 分享了她从纽约学表演、做狗美容师到转向文案写作、最终回归游戏叙事设计的曲折道路。两位嘉宾与主持人 William Dalthal 和 Tim Wood 讨论了在多份兼职工作与游戏开发之间平衡时间的策略、面对 Steam 愿望单仅 324 的现实心态、以及在财务压力和外界挫折中保持动力的方法。节目后半段还回答了社区关于抑郁、财务焦虑和生产力工具的问题。
