精华回顾：迷幻药物与神经刺激如何重塑大脑 | Dr. Nolan Williams

Stanford 精神科医生 Dr. Nolan Williams 与 Andrew Huberman 深入讨论了抑郁症治疗的新范式。Williams 提出「精神科 3.0」的概念——从早期的心理治疗（1.0）和化学失衡假说（2.0）转向聚焦大脑回路的治疗方法。他详细介绍了 Stanford Neuromodulation Therapy（SNT），一种利用 spaced learning theory 将六周 TMS 疗程压缩到五天的突破性方案，缓解率达 60-90%。对话还涵盖了 psilocybin 治疗抑郁症、MDMA 治疗 PTSD、ibogaine 用于特种部队退伍军人的道德创伤，以及 ayahuasca 的神经化学机制。Williams 强调这些 psychedelics 与 TMS 作用于相同的脑网络连接，暗示着一个趋同的治疗机制——即解耦负性情绪系统与自我表征之间的过度连接。

抑郁症：全球最致残的疾病

Andrew Huberman

在我们深入 psychedelics 之前，我想先聊聊抑郁症。我听 Nolan 在一个演讲里说过，抑郁症可能是全世界最具致残性的疾病，但相比癌症等其他疾病，我们的治疗工具却相当有限。

Nolan Williams

抑郁症确实是全世界最致残的疾病。有意思的是，它既是其他疾病的风险因素，又会让其他医学和精神疾病变得更严重。最近美国心脏协会把抑郁症加为冠状动脉疾病的第四大风险因素，跟高血压、高血脂、糖尿病并列。

脑心连接：从 TMS 到心脏

Nolan Williams

我们实验室做的很多工作是在测量脑心连接。我们可以用 transcranial magnetic stimulation 直接减慢心率，在情绪调节区域捕捉到这个心率减速。我们一直对最严重的急性抑郁症患者的治疗问题很感兴趣——那些进入急诊和住院病房的人。

这个领域一直没有持续有效的治疗方法，患者得到的还是跟门诊一样的标准口服抗抑郁药。我接受了神经内科和精神科的双重训练，看到神经内科有那么多治疗急性脑部问题的方法，就特别想在精神科复制这种模式。

Andrew Huberman

很多人觉得心脏和大脑的关系比较玄学，但你说的是实际的物理连接。

Nolan Williams

第一个接收刺激的地方是 dorsolateral prefrontal cortex，有点像大脑的控制中枢。利用 Faraday 定律，磁脉冲在脑组织中感应出电流，直接去极化皮层神经元。在 scanner 中能看到信号向下分布到 anterior cingulate、insula 和 amygdala，最终通过 vagus nerve 进入心脏。

心脏非常一致地是背侧前额叶皮层的终端器官。对视觉皮层或运动皮层做同样的事不会得到这些发现，这真的是特定于大脑控制区域的。Martin Arns 在荷兰的研究也验证了同样的连接，已被重复四五次。

TMS 如何恢复大脑的控制秩序

Nolan Williams

很多患者告诉我，他们的治疗师说他们在治疗中不够努力。这些是中度到严重的抑郁症患者。当我们用 TMS 五天方案让他们好起来后，他们会说：我整个周末都在看心理治疗书，现在我终于能理解了。

我把 TMS 看作一种外源性认知功能。在较轻的抑郁症中，心理治疗可以把 prefrontal cortex 打开让它管理深层区域。但在抑郁症里，关系是倒过来的——深层区域在管着 prefrontal cortex。一种情况像教练告诉球员该怎么做，另一种像球员告诉教练该怎么做。

抑郁症看起来是一堆半自主的自发内容，来自 cingulate 这个冲突检测系统。左侧 dorsolateral 没有足够地压制它。心理治疗恢复这种控制，TMS 则外源性地做同样的事——恢复左侧 dorsolateral 对 cingulate 的管控，这跟治疗改善直接相关。

TMS 几乎就像大脑的锻炼，用生理相关的信号反复锻炼这个区域。有意思的是，当患者在一周内足够早达到缓解后，我们有大概五六个人报告了类似的体验——他们去了海滩，完全活在当下，整整一个小时。他们说在正念书里读到过，但从未真正体验过。

SSRI 的定位：有效但机制不同

Andrew Huberman

我想在深入 ketamine 和 psilocybin 之前先聊 SSRI。我的理解是它们对 OCD 和抑郁症都有效，我们应该怎么看待它们？

Nolan Williams

SSRI 确实有效，很多 meta-analysis 已经证明了。在一个亚群体中，它们对抑郁症、OCD、广泛性焦虑障碍等都有很好的效果。问题在于它们不是立即生效的，说明可能不是 serotonin 直接驱动的，更可能是 brain plasticity 效果。

并不存在 serotonin 的缺乏，你不是天生就有化学失衡。精神科早就知道这个了，但最近才在大众媒体引起关注。

精神科 3.0：从化学失衡到回路校准

Nolan Williams

精神科 1.0 是 Freud 和心理治疗——聚焦于家庭经历和认知矫正。精神科 2.0 是化学失衡假说——说你缺少某种化学物质。两者都在告诉患者「我身上有什么缺失」。

TMS 和 psychedelics 传递的是不同信息。TMS 起作用了，没有任何 serotonin 进出大脑，而且我们的快速 TMS 在一到五天内见效。我们只是在激活脑区，关注的是回路——这就是精神科 3.0。

很多团队在做治疗前后的 neuroimaging，能看到 psilocybin 或 ketamine 在药物代谢后仍存在的回路变化。我们用 Stanford Neuromodulation Therapy 看到的 subgenual-default mode network 连接变化，跟 psilocybin 和 ketamine 作用于同一组脑区，暗示趋同机制。

它把故事重新聚焦在高度可纠正的东西上——本质是电生理学，是重新校准一个可以被校准的回路。精神科 3.0 的核心理念是：精神疾病是可恢复的，人们是可以好起来的。

很多患者告诉我：我不再害怕自己是永久性坏掉了，不再害怕化学失衡还在失衡，不再害怕童年经历会永远驱动这个问题。这就是这件事如此有力量的地方。

为什么大脑会抓住无益的规则不放

Andrew Huberman

高剂量 psilocybin 临床疗程中，患者报告在服药期间有东西自发浮现——看到老问题的全新视角。你觉得大脑为什么会抓住那些对我们没好处的规则不放？

Nolan Williams

这是一个进化神经生物学的答案。我们是一堆在现代社会不太有用的生物学的结果。以 PTSD 为例，退伍军人的那些反应在战场上是高度适应性的——听到巨响就躲起来，放回战场上这是救命的。但在家里完全没用。

但不管什么原因，MDMA 和 psilocybin 这些物质似乎能缓解这些症状，而且效果相当持久。在没有这些物质时症状会一直持续，但暴露后就能看到问题解决。我们在治疗海豹突击队时看到，他们回来说 PTSD 症状终于消失了。

MDMA 治疗 PTSD：临床数据

Nolan Williams

我的态度很实用——如果这种东西虽然带着文化包袱但最终被证明有治疗效果，而且临床试验能说服我，那绝对应该用。

Andrew Huberman

第一个 MDMA 试验发表在 Nature Medicine 上，大概说了什么？

Nolan Williams

MDMA 在一到几次疗程后有显著的抗 PTSD 效果。在临床环境下，标准 MAPS 剂量 150-175 毫克，大约三分之二的人有临床显著的 PTSD 变化。在早期试验中跟踪患者，有些人效果能持续好几年。

相比之下，单次 ketamine infusion 平均只维持约一周半。所以 psilocybin 治疗抑郁症和 MDMA 治疗 PTSD 的故事对我来说更有意思。

Psilocybin 的神经化学

Andrew Huberman

Psilocybin 治疗抑郁症的有效率是多少？

Nolan Williams

在 open-label 研究中大约一半到三分之二的人好转，在 blinded 试验中大概三分之一。

David Nutt 和 Robin Carhart-Harris 的 neuroimaging 工作是最早的。他们以为 psychedelics 下大脑活动会增加，结果发现相反——总体活动是下降的。但全局 connectivity 增加了，大脑从 small-world 模式转向更广泛的连接模式。

真正有意思的是，psilocybin 的抗抑郁效果涉及 subgenual anterior cingulate 和 default mode network 之间的连接性变化——跟我们 TMS 方法看到的完全一致。本质上是解耦了负性情绪系统与自我表征之间的过度连接。

Ibogaine：最强效也最高风险的 psychedelic

Andrew Huberman

关于 ibogaine——它在美国合法吗？谁在用它，治什么？

Nolan Williams

Ibogaine 是从非洲加蓬的 iboga 树根皮中提取的一种生物碱。服用者闭眼后会重新经历早年记忆，从充满共情的第三方视角看待过去的经历。它绝对不是娱乐性物质，人们称之为「人生回顾」或「一夜间的十年心理治疗」。

它持续 24-36 小时，是我所知最长效的 psychedelic。我们已经对前海豹突击队员、陆军游骑兵等特种部队人员进行了首次人体完整评估，包括临床量表、神经认知测试、neuroimaging 和 EEG。

Ibogaine 似乎是最有效的 psychedelic，但风险也最大，因为有心脏效应。不过可以通过心电图筛查降低风险。从反馈来看效果相当戏剧性。

士兵们会经历 moral injury——也许意外导致平民伤亡。他们回来后说原谅了自己，终于能用不同视角看待自己，对自己有了共情。这是巨大的突破。

Psychedelics 需要严格医学监督

Nolan Williams

如果你摆脱 Timothy Leary 时代的社会文化建构，纯粹从科学角度看——如果今天才发现这些物质，我们会说它们是精神科的巨大突破，因为它们结合了心理治疗和药物的效果，让人在一种特殊状态下重新审视问题。

但这些药物不能是娱乐性的，它们太强大了。如果连 SSRI 都需要处方和每周医生评估，那说这些物质可以在严格医学监督之外使用是荒谬的。这一代 psychedelic 研究者对此非常清楚。

Ayahuasca：两种植物的神奇组合

Nolan Williams

Ayahuasca 在巴西、秘鲁等南美国家作为圣礼使用。两种植物组合——一种含 DMT，一种含可逆 MAOI。单独使用任何一种都无效，但组合后 MAOI 防止 DMT 在胃肠道被分解，让它能跨过 blood-brain barrier。

亚马逊有一两万种植物，不知道怎么有人找到了这个精确组合。如果用标准不可逆 MAOI 替代，会导致 serotonin syndrome。这个甜点是 ayahuasca 实践者不知怎么找到的。

它看起来也很安全——UCLA Harbor 的研究甚至发现接触小剂量 ayahuasca 的儿童没有神经认知影响。在巴西某些天主教派别中，它已被用作宗教圣礼。

巴西监狱研究：Ayahuasca 与累犯率

Nolan Williams

巴西研究者给一半囚犯惰性物质，另一半 ayahuasca 疗程。接触 ayahuasca 的那组累犯率统计上显著低于对照组。这暗示着不管发生了什么，似乎对驱动犯罪行为的东西有影响。

我绝对不是说应该给囚犯 psychedelics，那是非常激进的事。但它确实提出了一个有趣的问题——到底是什么驱动人们改变行为。

SNT：五天内达到 60-90% 缓解率的突破

Nolan Williams

SAINT 或 SNT——Stanford Neuromodulation Therapy。TMS 设备递送的是一个治疗 protocol，即特定脑区的刺激参数集。我们要解决的是高急性精神科急诊状态下没有有效治疗的问题。

我们利用 spaced learning theory 来重新组织刺激。就像临考突击用卡片——每隔约一小时复习一次来优化学习。旧的 TMS 方法每天一次、每周五天、六周——就像每天只学一小点，效率很低。

我们把整个六周疗程重新组织成时间更密集的形式。同时发现人们一直在给 TMS 剂量不足。最终方案：五倍正常剂量，七个半月的量压缩到五天，每小时一次，每天十个小时。50 小时块中实际刺激 90 分钟。

结果是在一到五天内，60-90% 的人进入完全缓解。持久性可变——有人四年缓解，有人一年。信号本身很简单但深刻：开启，保持开启，记住保持开启。利用大脑自己的通讯方式来让它走出抑郁状态。

很酷的是，患者走出治疗后报告没有任何副作用，感觉恢复正常了。

Andrew Huberman

Nolan Williams

谢谢你。
