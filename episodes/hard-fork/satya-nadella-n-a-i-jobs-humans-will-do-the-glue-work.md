## 开场与 Microsoft AI 战略

**Kevin Roose**

让我们开始今天的节目。今晚我们的第一位嘉宾是我们一直非常期待的人。我们多年来一直想请他来节目，他终于同意了，今晚他就在这里。请大家热烈欢迎 Microsoft CEO Satya Nadella。

Satya，我必须先坦白一件事，那就是自 2023 年以来我一直不是 Microsoft AI 产品的常规用户，因为其中一个产品曾试图拆散我的婚姻——当然，我必须补充说明，它失败了，我的妻子就在那里。那是我们还没有考虑一个没有护栏的世界的时候。

但自那时以来，Microsoft 在 AI 方面发生了很多事情。对于那些可能有一段时间没有关注的人，你们最近在 AI 方面做了什么？

**Satya Nadella**

我认为我们正要从一个根本的转变开始——不再把 AI 当作单一的事物来谈论，而是要形成一幅关于 AI驱动生态系统的心理图景。

今天你可以想一想，从你第一次使用 Sydney 到现在，一直是关于前沿模型的问题。但如果我们要过渡到一个由 AI 驱动的经济体，它不能只关乎一个模型，不能只关乎三家公司。它必须是一种广泛感受到的变化——整个经济体处于前沿，而不是某个公司或某个模型处于前沿。

作为一家平台公司，这就是 Microsoft 正在做的事情。上周我们举办了开发者大会，核心主题就是：我们能否构建平台和工具，让每个国家、每个企业都能在前沿水平上运作？这才是关键问题。

如果说我的模型能做到这个，但经济增长只有 2%，这意味着事情不会有好结果，除非我们真正达到一个经济体经济增长出现拐点、且这种增长广泛分布的状态——因为前沿技术的好处必须普及。这正是电力和其他通用技术发生的情况。

## Project Solara：Agent-first 设备

**Casey Newton**

我想问一下上周 Build 大会上你宣布的一个平台转变——Project Solara。你说这将是 agent-first 硬件，下一代设备，上面会有 agent 运行。能多告诉我们一些吗？能举个例子说明 Project Solara 的设备是什么样的吗？

**Satya Nadella**

我们在 Build 上做了两件有趣的事。第一件是关于 PC 本身——实际上 Jensen 在前一晚的 Computex 上也做了类似的事情，他展示了那些台式机和笔记本电脑的照片。我最喜欢的一张照片就是 Jensen 拿着那些桌面设备，因为里面用的是 RTX 芯片，这是一个新的 SOC，本质上在 PC 上有一 petaflop 的算力。

这是关于把新功能带到旧的外形规格上。想想我所描述的「不计量的智能」——你能有一台 Windows 计算机，能本地运行万亿参数模型，我认为如果你要让 agent 24×7 运行，这是非常必要的。

但接下来的问题是：在一个有这些模型和长期运行 agent 的世界里，如果我是一名医院护士，从一个工作站走到另一个工作站，我能不能拿出一个设备，它可以扫描、可以输入、可以把我的语音输出转化成 prompt？这就是我认为 agent-first 设备的样子。

手机的中心地位对于很多我们使用的应用来说仍然会存在，但在 agent 世界里，你会拥有一种环境智能，就像一种感知场，然后与你的模型协作。这就是我们的目标：发明新的外形规格，不被旧的外形规格所束缚。

我们现在的挑战是思考如何在硬件和游戏两方面创新，同时做到经济上可行。Asha 刚接手 Xbox，她提出的一点是我们投入了很多——没有人能指责 Microsoft 过去 25 年没有投入。现在我们要把它变成一个可持续的业务，提供从根本上说是最好的娱乐来源之一。

我们的挑战是我们没有把这种娱乐货币化。事实上，我们一直在补贴这种娱乐。实际上 Xbox 游戏在 YouTube 上的货币化比在 Microsoft 这里还要多。但这并不意味着我们要做不合自然的事。我们要做我们的本职工作：构建好的游戏、构建好的硬件，但必须以经济可持续的方式来做。

Asha 上任 100 天了，她发了一篇文章说在接下来的 100 天里，她会重新审视，确保我们在硬件和发行两方面都能满足粉丝的期待。

**Casey Newton**

能给我们更多细节吗？比如当我听到这个，我会想：也许 Xbox 会变得更贵，游戏会变得更贵？能给游戏玩家什么甜头吗？

**Satya Nadella**

我们必须找到在经济上对客户和我们都有意义的方式来交付游戏。今天存在一个问题，不幸的是因为云和 AI 发生的事情，价格上涨了——PC、手机都受影响，Xbox 也受到影响。半导体供应的稀缺性，特别是内存，对消费电子产品整体产生了巨大影响。这是暂时性的，我认为我们会度过难关，这不是永久性的。

但有一个永久性的问题：Xbox 未来的模式是什么？PC 和主机都有各自的位置，移动端也有人在其他地方玩游戏。我们必须把这一切整合起来，同时保持我们一直做的事情。

## MAI 模型与前沿策略

**Casey Newton**

让我们谈谈那些模型。你的目标是打造世界上最好的前沿模型吗？如果是，超越 ChatGPT、Gemini、Claude 的策略是什么？

**Satya Nadella**

我们的真正目标是让生态系统中的每个人都能达到前沿。我们会采取稍微不同的思路。比如，想想如何构建一个前沿模型——你需要爬坡、做 RL，然后需要数据。但这时候数据已经饱和了，所以你基本上要从每个地方搜刮数据。

问题是，如果你反过来想：有一个基础模型有推理能力、有 agent loop，但你可以把它带入每个公司的 RL 环境。如果公司的未来是人力资本和 token 脚本，我希望每个公司的资产负债表和损益表都有这两项。这就是我们前沿模型的目标。

我们的模型应该是他们能使用的最好的基础模型，甚至保留权重——肯定是 harness 和 context，这些属于他们，他们可以用其他任何模型替换我们的模型。

这就是我认为的愿景。我总是问一个问题：为什么世界需要 Microsoft？如果我们成功，我们周围的世界能否也成功？我相信这是更可持续的方式。

## AI 反噬与公众形象

我们必须问一个关于全国范围内 AI 反噬的问题。毕业典礼演讲者被嘘，AI 的民意调查数据很差，很多人对数据中心不满。Microsoft 在这个问题上有什么角色？行业如何找到一条前进的道路，变得更受欢迎一些？

**Satya Nadella**

我认为我们可以从描绘一幅图景并交付结果开始——为什么每个人都是利益相关者。你不能出去说我有这个不可思议的技术，但你不会有工作，而且我们会拿走你所有的水和所有的能源，祝你好运。难怪会有这么多焦虑。

无论是学生还是社区，你必须做艰苦的工作。现在情况就是这样，你不能否认这种认知很差。我觉得 Microsoft 想要——就拿数据中心来说吧，很有意思。我们在华盛顿 Quincy 运营了 20 多年，我们刚刚庆祝了第 20 年。

如果我看这 20 年 Quincy 发生的一切：他们的税基增加了，当地税收下降了，当地就业增加了——因为有数据中心。实际上数据中心让 Quincy Washington 重获新生。我们做了烧烤活动，人们来庆祝这个城镇因为我们的存在而复兴。这是 20 年的第一个纵向案例。

这就是数据中心所在社区应该做的：不能提高能源价格，要非常谨慎，实际上应该补充他们使用的所有水，并创造经济机会。

在整个经济体中，如果小企业觉得 AI 让他们更高效，如果每个大型跨国公司能说我正在构建 token 脚本和人力脚本——因为关键问题是就业。每个人都认为所有工作都会消失。但如果你仔细想想——我不是说不会有真正的职位替代，工作流程不会改变。

## AI 对就业的影响

**Satya Nadella**

就拿软件开发来说，现在当然完全是 agent 化了。但想想 GitHub 应用的演进——当我有一百个 CLI 时，我需要什么？我需要一个新 IDE，叫做 ADE，一个后端，某种帮助我管理所有复杂性的软件。

我认为我们必须思考新的工作，这会是元认知、元工作，这会有工资，我们必须对此具体说明。

**Casey Newton**

但帮我们理解一下你对潜在颠覆的看法。过去几个月我跟很多经济学家和技术领袖谈过，意见真的各不相同。有些人说我明年肯定会少雇人，因为 AI。也有人说我不够工程师，我想要更多。你在这个问题上站在哪里？两年后你会更多工程师还是更少？

**Satya Nadella**

如果在 80 年代初有人来跟我们说，世界上会有 35 亿人，他们都会是打字员，我们会说为什么世界需要 35 亿打字员。但事实上我们有，我们每天早上起来都在打字，但我们做的是所谓的信息工作、知识工作等等。

这种变化必须发生，每个人都会有新的名称，会有工资支持这种重塑。过去的软件开发者到未来的软件开发者可能有类似的技能，但他们做的工作不同，因为他们管理着一百个 agent、一千个 agent。

我有一个同事提出了一个很美的概念。就像软件开发中我们有测试覆盖率的概念，我们现在学到的新东西是我所说的认知覆盖率。软件开发者做什么？我有一个由 agent 写满的代码仓库，我在认知上理解发生了什么，我需要工具来获得认知覆盖率——这就是软件开发者的工作。

为了做到这点，你需要上学，学习计算机科学，获得认知覆盖率。这种工作的重塑——工作产物、工作流程——因为软件从输入到输出，这是格式变化、产物变化，工作流程随之改变，工作也随之改变。

**Kevin Roose**

我觉得人们真正想听到的是某种组合——你的工作不会变化太多，或者如果变化了，你会得到更高的报酬。你觉得对大多数人来说这两件事会成立吗？

**Satya Nadella**

我认为这是关键。工资一直是关于我们作为社会看重什么。过去 200 年、250 年的增长是关于某种形式的专业知识和知识积累。当你有某种专业知识的丰富供给时，人类能构建新的不可训练的专业知识的能力是什么？

今天早上我读了 Sarah Guo 的一篇博客，我觉得很有意思，她提到什么是不可训练的部分，这适用于组织，也适用于我们自己。作为人类，我们有 agency、有 ambition，这不应该被低估。

如果你看今天的人力资本是什么，尽管我们有所有这些数字化系统，我们仍然在做胶水工作。我们会发现新的胶水工作，与所有这些自动化一起发生——我认为这就是变化的过程。

## AGI 与 10% GDP 增长

**Kevin Roose**

你在 2025 年 2 月告诉 Dwarkesh Patel，你对实现 AGI 的基准是 10% GDP 增长。看起来我们现在离 10% 还不是很近。我好奇你现在如何看待你一年前说的那句话，你看到最近的加速让你觉得这更可能了吗？

**Satya Nadella**

事实上这是我经常思考的问题——一个非常强大的通用技术及其扩散的困难程度，以及所需的管理变革。比如现在我们接下来一两年面临的挑战之一是 token 的经济学。

残酷的事实是，生产力提升的边际成本必须与 token 的边际成本相匹配。这是一种管理纪律。你不能只说我喜欢 token maxing，因为这像是把钱存到我的银行里。业务必须从中受益，这才是真正驱动它的因素。

实际上这很有意思。10% 增长的方程式是当 token 的边际成本与边际价值完美匹配并且定价合理时。如果这发生，10% 绝对会发生。但现在大家都在 wipe code 和 token maxing，那不是实现 10% 增长的方式。

**Casey Newton**

Microsoft 内部有多少 token maxing 进行过？你有没有——

**Satya Nadella**

我也深陷其中。我的意思是，我也喜欢 token maxing，所以我也上瘾了。就像我喜欢这个东西。但当新奇感消退时，你必须退一步想：我到底想创造什么？

事实上我现在喜欢 Copilot 的 auto mode。我们现在有一个很好的经济模型来支持它。基本上我会说：不要用前沿模型解决非前沿问题。请让我们把这些东西匹配起来，这样你能获得输出和经济效益，而不是一场只做不增加价值的事情的竞赛。

**Kevin Roose**

给我们一点 Satya 的 token maxing 的感觉——你最近有哪些大的 token 项目？

**Satya Nadella**

最近我构建的一个东西：我一直觉得我想要一个仓库，能随时与我没有参与的讨论保持同步。想想这个概念——我喜欢这个，因为今天这其实是不可能的。

## AI 的政治经济学

**Kevin Roose**

你一直在思考 AI 的政治经济学。显然你谈到的扩散、采用和 GDP 增长都是其中一部分。你认为在 San Francisco 领导 AI 公司的人们对 AI 的政治经济学有什么误解？

**Satya Nadella**

我不会说他们错了。但我回头看时，去年十二月我读了一本很酷的书，是 Joel Mokyr 和几位合著者写的，我觉得叫《Parallel Paths to Prosperity》。他们描述了过去一千年西方如何增长，以及中国发生了什么。这是一千年历史。

我从那本书中学到的基本东西是，西方把三件事形成了一个良性循环：技术革命、市场和民主，彼此互相制约。这就是为什么不存在单纯的经济，而是政治经济学。民主最终控制市场发生什么，然后技术试图颠覆两者，然后你保持制衡。

这是神奇的，是世界出现过的最不可思议的社会建构之一。这成为了模式，我们现在需要为这个时代重新定义同样的模式，但它会运作，因为它上次运作了。

因此我认为我们需要提醒自己这种平衡和相互制衡——无论是在 San Francisco、Washington DC，还是其他任何地方，我们都应该追求这一点。

## AGI 的看法

**Casey Newton**

最后一个问题，我还在试图弄清楚 Kevin 可能会说的——你有多 AGI pilled。在 Silicon Valley 有一种感觉，这次真的不一样，锯齿状的前沿会永远前进，突然间今天 AI 能自动化的小任务会转化成完整的工作。你买这个故事多少？

**Satya Nadella**

我认为凡是能闭环的事情——比如编码，实际上 AI 研究也是可能闭环的——我认为我们现在有足够的证据证明这点。但这够吗？我不这么认为。

当人们谈论这项任务可验证性如何时，在知识工作的混乱真实世界里，只说我看人类活动的痕迹就够了来闭环——我不这么认为。这我认为是挑战。

当我在一个会议里，我说事情、我记事情、我观察事情，但我怎么处理这些，今天这不是一个我可以 rally 的痕迹。这就是我们在低估人力资本中不可验证的部分。

所以我相信进展会持续发生。我仍然更多处于这样的世界：这是平台和工具，非常强大，非常颠覆。

我有很多谦卑——很多事情会改变，但归根结底，电力是这样，蒸汽刚出现时也是这样。所以我不会坐在那里想这是我们发明的最后一个技术。我不买这个。

我觉得在所有技术的殿堂里，这是一个大步骤。但我认为还会有更多。
